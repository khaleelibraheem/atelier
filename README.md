# Atelier. | Minimalist E-Commerce

Atelier is a high-performance, minimalist e-commerce storefront built with Next.js 15, Tailwind CSS, and React Context. It features a curated selection of furniture and home accessories with a focus on typography, motion, and user experience.

![Atelier Preview](/public/preview.png)

- **Storefront:** Editorial-style homepage with infinite marquees, grid layouts, and sticky positioning.
- **Dynamic Routing:** Product and Collection pages generated from a centralized data store.
- **State Management:** Fully functional Shopping Cart and Checkout flow using React Context.
- **Performance:** Optimized images using `next/image` with AVIF/WebP formats.
- **SEO:** Dynamic metadata generation for every product page.
- **UX Details:** Mobile-first drawers, skeleton loading states, and simulated payment processing.
